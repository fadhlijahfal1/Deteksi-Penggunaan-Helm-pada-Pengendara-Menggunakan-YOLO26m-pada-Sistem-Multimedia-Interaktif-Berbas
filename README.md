# Deteksi Penggunaan Helm pada Pengendara Menggunakan YOLO26m Berbasis Transfer Learning
Repositori ini berisi kode sumber dan dokumentasi untuk proyek deteksi penggunaan helm pada pengendara sepeda motor. Proyek ini memanfaatkan algoritma Deep Learning untuk memproses dan menganalisis citra pengendara. Untuk meningkatkan efisiensi pelatihan dan akurasi model, sistem ini mengimplementasikan teknik Transfer Learning menggunakan arsitektur pre-trained model YOLO. Proyek ini bertujuan untuk mendukung sistem keselamatan lalu lintas pintar (Intelligent Transportation Systems) dengan mengidentifikasi pelanggaran tidak memakai helm secara otomatis.
